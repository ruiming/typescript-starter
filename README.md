## TypeScript-starter

1. 使用 mocha 作为测试框架

2. 使用 chai 作为断言库

3. 使用 Angular Git Commit Message 规范

4. 使用 TSLint 和 tslint-config-standard 进行代码风格约束

5. 使用 husky 进行 lint 检查和 commit msg 检查

6. 使用 nyc 生成测试覆盖率报告

7. 使用 nodemon 作为开发启动器

8. 使用 yarn 和 npm5 构建

9. 默认开启 TypeScript Decorator 支持

10. 配置模块解析, src 可以作为 app 模块使用
